# tokenomnom

<p align="center"><img src="assets/hero.png" alt="tokenomnom — your agents nom tokens. this shows the bill." width="840"></p>

tokenomnom reads local Codex and Claude Code logs and shows token usage as API
list-price equivalents, not actual bills. It ships as `tokenomnom` and `nomnom`.
